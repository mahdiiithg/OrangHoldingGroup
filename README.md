# Orang Holding Group

Corporate website built with [Astro](https://astro.build).

## Commands

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # output → dist/
npm run preview
```
H.

## Vercel

Root directory: **`.`** (repo root)  
Framework: **Astro**  
Output: **`dist`**

## Structure

```
src/          pages, components, layouts, data, styles
public/       static assets (images, robots.txt, sitemap)
```
